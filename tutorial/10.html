Part 10 - HTML Content

One of the great things about PlusSlider is that you're not limited to just images for the slides
-- you can have any kind of HTML element be a slide (for example, a <div> with more HTML inside).

An important thing to keep in mind when dealing with non-image slides
is that you must explicitly set a width and height on each slide in CSS.
The easiest way to do this is to put a width and height on the ".plusslider .child" selector.

If you want to add margin or padding to a particular slide, you'll need to either
reduce the width/height on that slide (thus overriding the general width/height on
".plusslider .child"), or add another wrapper element inside the slide contents
and put the margin/padding on that inner element.
Examples of both techniques can be seen in this part's CSS.
